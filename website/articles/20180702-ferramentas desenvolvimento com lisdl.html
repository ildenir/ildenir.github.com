Desenvolvimento com autotools

As ferramentas automake e autoconf suportam uma infraestrutura de compilacao automatizada que facilita em muito a vida do desenvolvedor ao gerar makefile portaveis. O uso eh muito facil basta um pouco de pratica e uma receita de bolo.

Inicialmente, os codigos fonte do aplicativo devem ser criados. Em seguida, o processo de compilacao deve ser descrito em um arquivo Makefile.am que serah processado para gerar um arquivo de entrada para o script configure. O arquivo configure eh gerado pelo autoconf. Daih, a compilacao da aplicacao segue o padrao : configure &amp;&amp; make &amp;&amp; make install .

Suponha que foi desenvolvido o codigo fonte main.cpp. Para criar um makefile para o fonte, escrevemos o arquivo Makefile.am como segue: 

bin_PROGRAMS = teste
teste_SOURCES = main.cpp

Agora, execute o programa autoscan para gerar um template do arquivo de configuracao.

Execute cat configure.scan &gt; configure.ac

Edite configure.ac como segue:

-- Autoconf --

ACPREREQ(2.61)

ACINIT([teste], [0.0], [ildenir@gmail.com])

AMINITAUTOMAKE

ACCONFIGSRCDIR([main.cpp])

#ACCONFIGHEADER([config.h])

ACPROGCXX

ACCONFIGFILES([Makefile])

ACOUTPUT

Execute aclocal; touch README, NEWS ; automake. Caso esteja faltando algum arquivo execute automake –add-missing

Execute autoconf e pronto.

Para finalizar, compile a aplicacao com a sequencia de comandos configure &amp;&amp; make &amp;&amp; make install

Para utilizar SDL escreva o makefile como segue:

SDLLIBS= `sdl-config –libs`

SDLINC = `sdl-config –cflags`

INCLUDES = $(SDLINC)

binPROGRAMS = teste

testeSOURCES = main.cpp

testeLDFLAGS = $(SDLLIBS)

Projetos SDL com AutoTools

Vai um jeito mais simples de configurar as ferramentas autotool para projetos sdl. Suponha que temos um arquivo fontes main.c a ser linkado com a biblioteca sdl. Escreva o Makefile.am como segue:

AUTOMAKEOPTIONS = foreign

binPROGRAMS = sdldemo

sdldemoSOURCES = main.c

o arquivo configure.ac como:

ACPREREQ(2.57)

ACINIT([sdldemo], [0.0.0],[ildenir@gmail.com])

AMINITAUTOMAKE

ACCONFIGSRCDIR([main.c])

#ACCONFIGHEADER([config.h])

ACPROGCC

SDLVERSION=1.2.5

AMPATHSDL($SDLVERSION,:, ACMSGERROR([*** SDL version $SDLVERSION not

found!]))

CFLAGS=" $CFLAGS $SDLCFLAGS"

LIBS=" $LIBS $SDLLIBS "

#ACSUBST(CFLAGS)

#ACSUBST(LDFLAGS)

ACHEADERSTDC

ACCHECKHEADERS([stdlib.h])

ACHEADERSTDBOOL

ACCHECKFUNCS([atexit])

ACCONFIGFILES([Makefile])

ACOUTPUT

Observe que a intrucao AMPATHSDL faz todo o trabalho de configuracao do ambiente.
